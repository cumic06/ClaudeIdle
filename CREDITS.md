All assets are free/open license, safe for personal and commercial use.

- **`assets/sprites/pet.png`** — the player character. Original pixel art, hand-authored to match the Claude Code app icon (not a third-party asset). Accessory overlays (`acc-*.png`) are drawn in the same style to line up on top of it.

Design: the player is always the Claude Code mascot — it never changes shape or color. Leveling up unlocks accessories (glasses, headphones, crown, cap, a rare holographic aura) that layer on top of it. The found CC0 assets are cast as enemies you catch for EXP, not as the player's look.
